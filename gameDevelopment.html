    Game projects 

 

Below are some game projects that i have made in High school and one from my college. Screenshots and videos will be shown below with a short description on what the game is a bout. There are also feature lists so you can see what exactly has been implemented into the game thats shown in the gameplay videos.             

Charmanders adventure:  

Charmanders adventure is my final project for my first unity class. It was a simple proof of concept demo where you could walk around and fight enemies, collect coins and go inside buildings. it was inspired heavily from the original zelda like rpg games. I just wanted to try to re-create something like that in the Unity engine.  

Features: 

  •  Top down camera view
  • Classic RPG combat
  • custom collectables and refillable health
  • breakable pots with randomized loot pool
  • enemy states, sleeping and follow mode
  • transitions between areas 
  • colorful sprites and UI 

Yoshis Prison break

Yoshis prison break is a single player puzzle game where you control two different characters to solve unique puzzles and escape the prison. It was made in Game maker studio 2

Features

  •  2D side scrolling perspective
  • 2 Playable characters
  • shootable eggs to break obstacles
  • disapearing platforms
  • Unique 2 character puzzles
  • character switching mechanic 

Charmander Commits Deforestation  

Charmander commits deforestation was created as a game for a project that needed to have a nature theme so we made a game about destroying it in our group. To progress in this game you must destroy all the trees and bushes to progress further along. there are two biomes, a field and an arctic plain. also featuring two challenging boss fights.  

Features

  •  2D top down action
  • Chaotic / comical atmosphere
  • Mouse aim
  • various enemy types with different health pickups
  • Chili pepper powerup. Increases player dmg and projectile speed
  • 2 playable biomes 
  • 2 Unique boss fights
  • Yeti boss arena features turret enemy system
